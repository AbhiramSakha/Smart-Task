
🧠 Smart-Task

A simple, responsive task management web application built with HTML, CSS, and JavaScript. This project lets users create, view, edit, and organize daily tasks efficiently.

🚀 Features

✔ Add new tasks
✔ Mark tasks as complete
✔ Responsive layout for desktop & mobile
✔ Styled UI with custom CSS
✔ Lightweight, fast, and easy to extend

📁 Project Structure
📦 Smart-Task
├── 📄 index.html        # Main user interface
├── 📂 style             # CSS files for design
├── 📂 js                # JavaScript for app logic
└── 📂 .vscode           # Editor settings

🛠️ How It Works

Open index.html in any browser to launch the app.

Use the input form to add a task.

Tasks are displayed on the page and can be managed interactively.

The JavaScript handles task creation, removal, and status updates dynamically.
